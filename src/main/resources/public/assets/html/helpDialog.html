Welcome to ScheduSmart!

This application allows you to find the best class schedule based on your time preferences. Here is how to get started:

  • Open the left search menu and select your term and courses.
  • Use the 'hide sections' buttons to ignore certain section types.
  • Use the 'preferred class time' section to prioritize classes before or after noon.
  • Use the 'day length preference' section to choose between more class per day (and thus more days off) or less.
  • Select 'hide closed sections' to ignore sections that are full or otherwise closed.
  • Press 'Search' to generate schedules.
  • Use the 'Prev' and 'Next' buttons to navigate generated schedules (ordered from best to worst).
  • Use the view button at the top-right of the page to switch between week view and list view.

Tip: Too many schedules generated? Try ignoring section types (under 'ignore sections') that have a lot possibilities, like TUTs.

Note: Online courses are always ignored.
